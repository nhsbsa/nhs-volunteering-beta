Advertise your volunteering opportunities

You can use this service if you want to:

  • advertise your opportunities and receive applications though NHS Volunteering
  • advertise your opportunities and link to application forms on existing systems

You can advertise either:

  • one or more specific opportunities
  • a generic listing for all your opportunities
Information:

We refer to 'specific opportunity' to describe an opportunity with a defined title, requirements and responsibilities.

A 'generic opportunity' is a broader listing that doesn't provide as much detail and it can include a range of opportunities.

The process

The information you enter will create a listing that potential volunteers will see when they search for an opportunity.

To complete a listing, you will need to complete the following sections:

1. add a preview

This is what potential volunteers will see first when they search for opportunities.

2. fill in the listing

You will add:

  • a summary
  • details of the type of person you are looking for
  • an overview of your application process
  • the availability you need
  • the contact details where you want to receive queries about the opportunity

3. add locations

You can add one or more locations for the opportunity. This will allow potential volunteers can find opportunities that are closest to them.

You will also be able to select if this is a remote opportunity.

4. select the application settings

You can decide if you want to receive applications through this service or through an external link.

You will set a closing date for this listing.

If you receive applications through this service, you can also set a limit to the number of applications.

Reviewing and making changes

You can save your progress and continue once you have entered a title.

You can also preview the listing before publishing.

Once you publish the listing, you can still change the closing date and the number of applications you want to receive.

Guidance

We have added examples and guidance to support you through each step of this task.
